# 🔥 INKED — Tattoo Studio Website Template

A modern, cinematic tattoo studio website template built with HTML, CSS, and JavaScript.
Designed for tattoo artists, studios, and creative professionals who want a premium online presence.

## ✨ Features

* 🎨 Modern dark premium UI design
* 🎬 Smooth animations (scroll + hover effects)
* 📱 Fully responsive (mobile, tablet, desktop)
* 🖼️ Tattoo gallery / portfolio section
* 📞 WhatsApp, call, and email integration
* 📍 Google Maps location integration
* ⚡ Fast loading & optimized performance
* 🛠️ Easy to customize (text, images, links)

---

## 🧱 Sections Included

* Hero Section
* About Section
* Tattoo Gallery
* Services
* Contact / Booking
* Footer

---

## ⚙️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP (for animations)
* ScrollTrigger
* Lenis (smooth scrolling)

1. Download or clone the repository
2. Open `index.html` in your browser
3. Edit content (text, images, links)
4. Customize as needed

* Edit text → `index.html`
* Update styles → `style.css`
* Modify animations → `script.js`
